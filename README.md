# Public Service Appointment Booking System

## Overview
A responsive web application that allows citizens to book appointments for public services.

## Technologies Used
- HTML
- CSS
- JavaScript

## Features
- Appointment booking form
- Date and time selection
- Duplicate booking prevention
- Persistent storage using LocalStorage
- Responsive design
- Cancel appointments

## Purpose
This project demonstrates frontend logic and user-focused design aligned with public sector service systems.
